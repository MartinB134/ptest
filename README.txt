Hot to install:
* PHP 5.3 required
* Unzip this package directly or to a subdirectory of your webroot (the latter is untested)
* Create a database
* Import the database dump (db.sql)
* Set the database credentials in index.php

How to do it:
* Go to index.php, read the tasks on the start page (index.php) and the tipps below CAREFULLY
* Do the test
* Dump your DB to db.sql
